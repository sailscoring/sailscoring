## Acceptance Criteria Template

For each high-priority story, define acceptance criteria:

```
### [Story ID]: [Story Title]

**Given** [precondition]
**When** [action]
**Then** [expected result]

**Given** [another precondition]
**When** [action]
**Then** [expected result]
```
