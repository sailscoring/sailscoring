ECHO Analysis Explanation

;

Each entry consists of the following

  • The handicap to be used for the following race
  • The difference from the handicap used in this race
  • The handicap actually achieved by the boat in this race, also known as the ECHO Index.
  • The elapsed time improvement needed to win, or NF if the boat was not a finisher
The final column shows
  • The change in handicap over the whole series
  • The median ECHO index for this boat

For a detailed explanation of ECHO handicaps  see this FAQ.
